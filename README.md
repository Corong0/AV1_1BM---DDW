# Lista de Tarefas

Uma aplicação de lista de tarefas moderna e interativa com um tema escuro elegante e estética ciano/roxo.

## 🎯 Funcionalidades

- ✅ **Adicionar Tarefas** - Adicione rapidamente novas tarefas à sua lista
- ✏️ **Editar Tarefas** - Modifique o texto da tarefa diretamente da lista
- 🗑️ **Deletar Tarefas** - Remova tarefas concluídas ou indesejadas
- ⚠️ **Validação de Entrada** - Impede adicionar tarefas vazias com mensagens de erro claras
- 📱 **Design Responsivo** - Funciona perfeitamente em dispositivos desktop e móveis
- 🎨 **Interface Moderna** - Fundo com gradiente escuro com acentos vibrantes em ciano e roxo

## 🚀 Como Começar

1. Clone ou baixe este repositório
2. Abra `index.html` em seu navegador web
3. Comece a adicionar suas tarefas!

## 📋 Como Usar

1. **Adicionar uma Tarefa**: Digite sua tarefa no campo de entrada e clique em "Adicionar Tarefa" ou pressione Enter
2. **Editar uma Tarefa**: Clique no botão de edição ao lado da tarefa que deseja modificar
3. **Deletar uma Tarefa**: Clique no botão de exclusão para remover uma tarefa da sua lista

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica de marcação
- **CSS3** - Estilos, gradientes, animações e layout responsivo
- **JavaScript Vanilla** - Manipulação do DOM e lógica de gerenciamento de tarefas

## 🎨 Destaques do Design

- **Paleta de Cores**: Fundo com gradiente escuro (#1a1a2e para #0f3460) com acentos em ciano (#00f5ff) e roxo (#9d4edd)
- **Tipografia**: Família de fontes monoespaciais (Courier New, IBM Plex Mono)
- **Animações**: Transições suaves e efeitos de deslizamento
- **Estados de Foco**: Feedback visual aprimorado com efeitos de brilho em elementos interativos

## ✨ Detalhes das Funcionalidades

### Gerenciamento de Tarefas
- As tarefas são armazenadas em um array JavaScript e renderizadas dinamicamente
- Atualizações em tempo real da lista ao adicionar, editar ou deletar operações

### Validação
- Detecção de tarefas vazias com mensagens de erro amigáveis
- Mensagens de erro desaparecem com animações suaves

### Experiência do Usuário
- Campo de entrada foca automaticamente após adicionar uma tarefa
- Efeitos hover nos botões para clareza de interatividade
- Estilos de botões responsivos com fundos gradientes
